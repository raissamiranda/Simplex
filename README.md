# Simplex

Simplex algorithm is used to solve the Linear Programming (LP) optimization problems.

## Description

The simplex algorithm operates on linear programs in the [[canonical form]]

:maximize <math display="inline">\mathbf{c^T} \mathbf{x}</math>
:subject to <math>A\mathbf{x} \leq \mathbf{b}</math> and <math>\mathbf{x} \ge 0</math>

with <math>\mathbf{c} = (c_1,\, \dots,\, c_n)</math> the coefficients of the objective function, <math>(\cdot)^\mathrm{T}</math> is the [[matrix transpose]], and <math> \mathbf{x} = (x_1,\, \dots,\, x_n)</math> are the variables of the problem, <math>A</math> is a ''p''Ã—''n'' matrix, and <math> \mathbf{b} = (b_1,\, \dots,\, b_p)</math>.

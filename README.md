# Simplex

Simplex algorithm is used to solve the Linear Programming (LP) optimization problems.

## Description

This project solves linear problems in canonical form:
> max $c^Tx$
> 
> subject to $Ax \le b$



$$
A = \left[
\begin{array}{ccc}
1 & xyz & a_{13} \\
25 & 0 & \cos{x} \\
0 & a_{32} & \Box
\end{array}
\right]
$$
